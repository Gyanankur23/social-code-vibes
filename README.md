A curated collection of interactive, visually stunning coding projects designed for social media sharing and engagement.

## 🎯 About

This repository hosts creative, production-ready web projects that are perfect for:
- Social media sharing

## 🚀 Projects

### Love Story
An interactive 3D love story experience with particle effects, animations, and engaging interactions.

**Features:**
- Smooth card transitions
- Particle effects
- Present opening animations
- Fully responsive design
- Touch-optimized for mobile
- Production-ready performance

**Tech Stack:** HTML, CSS, JavaScript (no dependencies)

## 🛠️ Adding New Projects

1. Create a new folder in `projects/`
2. Add your project files
3. Create a project-specific README
4. Commit and push
